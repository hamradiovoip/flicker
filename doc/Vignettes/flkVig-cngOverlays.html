Flicker Vignette - How do I change the image overlays preferences?

This vignette shows how do you can change the image overlays preferences.

Method

  1. You can reset the colors, view overlay options, and canvas size using the (Edit | Reset default view)
  2. You can change the overlay for the left, right and flicker windows and measured spots using the (View | Set view measuremens options | ...) options.
