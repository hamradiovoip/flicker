Flicker Vignette - How do I save transformed or overlay images?

This vignette shows how do you can save transformed or overlay images.

Method

Save the transform image

  1. Select the (File | Save transformed image) command. This saves the transformed image in tmp/ as a .gif with the same name of the input image.

Save the overlay image

  1. Select the (File | SaveAs overlay image) command to save the overlay image as a .gif image. You must specify the .gif image file name.
