Flicker Vignette - How do I load an active map image from the Web?

This vignette shows how do you can load an active map image from the Web.

Method

  1. Select the image you want to replace with an active map image
  2. Select the active map image with (File | Open active map image | ...) command. This will replace the selected image with the Web image. It also enables your access (in step [3]) to the Click to access DB checkbox.
  3. Enable the Click to access DB checkbox when you want to access the active Web site.
