Flicker Vignette - How do I reset the state for current images?

This vignette shows how do you can reset the state for the currently loaded gel images.

Method

  1. Select the (File | Reset images) command.
  2. This will remove the transformed and brightness/contrast images and any landmarks you may have defined for both the left and right image. It will not change your original preferences.
