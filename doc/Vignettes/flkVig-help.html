Flicker Vignette - How do I on-line help?

This vignette shows how do you can get on-line help from the Flicker Web site.

Method

The reference manual and vignette examples etc. are available on the http://open2dprot.sourceforge.net/Flicker Flicker Web site.
  1. You can look at the reference manual on the Web site.
  2. If you are running Flicker, you can access the reference manual and other parts of the Web site from the Help menu.
  3. Look at the list of Vignettes in the reference manual.
